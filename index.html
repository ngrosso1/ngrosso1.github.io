XV6 MLFQ Addition

Modified the original base code of the XV6 operating system to utalize an MLFQ (Multilevel feedback queue). This would give priority to a process that had been starved. The first queue (lowest priority) will promote a process to the second queue if its wait time in the queue exceeds 1 second. In the second queue (highest priority) the default RR (round robin) scheduling is utalized.

Path Finding Algorithm

Remotly worked in a group of 3 to test the difference between the path finding algorithms A* and Dijkstra. Written in C++ we utalized random graphs to show that A* is a improvment to Dijkstra's algorithm.
